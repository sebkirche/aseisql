Create a report to show changes in i/o statistics and fault that took place between two dbcc operations.

sp_dbcc_differentialreport [dbname, [ objectname]], [checkstorage] [,date1 [, date2]]

Parameter information :
dbname : The name of database for which results are available.
objectname : The name of a table or index to show results for.
date1 : The first date of a result.
date2 : The second date of a result.
