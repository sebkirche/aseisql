List expansion rules for a database.

sp_dbextend command, "database" [, db_devname [, arg1 [, arg2]]]

Parameter information :
command : Controls the detail level of listing. Possible values: list : Brief listing, listfull : Full listing
db_devname : The name of a database.
arg1 : The name of a segment.
arg2 : A special option to control the ordering of the output.
