Creates a login to use when connecting to another server with CIS.

sp_addexternlogin server [, loginame] , externname [, externpasswd] [, rolename]

Parameter information :
server : Name of the remote server.
loginame : Name of the login that can connect to the remote server. When loginame is not specified (null) all users connecting to the remote server will use the same external login and password combination (when there is no specific external login defined for the user).
externname : Name of the login on the remote server.
externpasswd : Password for the login on the remote server.
rolename : The name of the role the user is given upon log on the the external server.
