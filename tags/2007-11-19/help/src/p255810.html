Show resource limits.

sp_help_resource_limit [name [, appname [, limittime [, limitday [, scope [, action [, verbose]]]]]]]

Parameter information :
name : Name of a login or null for all logins.
appname : Name of an application or null for all applications.
limittime : The time during which the limit is active, or null for all times. Specify time with the following format "hh:mm".
limitday : Day during which the limit should be active. Specify the day of the week as defined within the default language on the server. Specify null for all days.
scope : Scope of the limit. Possible values: 1 : Queries, 2 : Query batches, 4 : Transactions, 6 : Query batches and transactions., null : All scopes.
action : The action for the resource limit. Possible values: 1 : Warning, 2 : Abort query batch, 3 : Abort the transaction, 4 : Kill the session, null : All actions.
verbose : Show more output Possible values: 0 : Show normal output (default behaviour)., 1 : Show more output.
