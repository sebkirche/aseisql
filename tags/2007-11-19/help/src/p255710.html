Configures caches.

sp_cacheconfig [cachename [, parm1] [, parm2] [, parm3]] [,parm4]

Parameter information :
cachename : Name of the cache.
parm1 : Specify the size of the cache in values of the following sizes : Possible values: G : GigaBytes, K : KiloBytes, M : MegaBytes, P : Pages
parm2 : The type of cache. Possible values: logonly : The cache will be used for transaction logs., mixed : Log and / or normal pages.
parm3 : Replacement policy. Possible values: relaxed, strict
parm4 : Speficy the number of partitions to be created for this cache. Format to be used is "cache_partition=x" where x is one of the following: Possible values: 1, 16, 2, 32, 4, 64, 8
