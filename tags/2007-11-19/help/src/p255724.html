Show information on columns in one or more tables or views.

sp_columns table_name [, table_owner] [, table_qualifier] [, column_name]

Parameter information :
table_name : Name of the table or view. Wildcards are supported.
table_owner : Owner of the table, wildcards are supported. When not specified first the current user name is used and then dbo to look for a table.
table_qualifier : The name of the database. Can only be null or the name of the current database.
column_name : Name of a column within the table or view. Wildcards can be used.
