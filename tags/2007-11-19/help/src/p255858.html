Allows a command to be executed on a remote server.

sp_passthru server, query, errcode output, errmsg output, rowcount output [, arg1 output [,arg2 output [, argn output]...]]

Parameter information :
server : Name of the server to execute the query on.
query : The query to execute.
errcode : The errorcode returned by the remote server.
errmsg : The error message returned.
rowcount : Number of rows affected.
arg1 : The first column from the last row returned from the last command in the query.
arg2 : The second column from the last row returned from the last command in the query.
argn : The n-th column from the last row returned from the last command in the query.
