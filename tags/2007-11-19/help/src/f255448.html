Returns statistical data for the specified object and index.

derived_stat({object_name | object_id}, {index_name | index_id}, "statistic")

Parameter information :
object_name : The name of the table you are interested in. If you do not specify a fully qualified table, derived_stat searches the current database.
object_id : Is an alternative to object_name, and is the object id of the table you are interested in. This must be in the current database.
index_name : Is the name of the index, belonging to the specified table that you are interested in.
index_id : Is an alternative to index_name, and is the index id of the specified table that you are interested in.
statistic : The derived statistic to be returned. Possible values: data page cluster ratio : The data page cluster ratio for the object/index pair, data row cluster ratio : The data row cluster ratio for the object/index pair, dpcr : The data page cluster ratio for the object/index pair, drcr : The data row cluster ratio for the object/index pair, index page cluster ratio : The index page cluster ratio for the object/index pair, ipcr : The index page cluster ratio for the object/index pair, large io efficiency : The large io efficiency for the object/index pair, lgio : The large io efficiency for the object/index pair, space utilization : The space utilization for the object/index pair, sput : The space utilization for the object/index pair
