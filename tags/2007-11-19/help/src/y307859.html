Modify the rule for a database

sp_dbextend "modify","database", db_devname, arg1, arg2, arg3

Parameter information :
db_devname : The name of a database or 'default' for a server-wide setting.
arg1 : The name of a segment in the database.
arg2 : Indicates the value you like to change. Possible values: "growby", "maxsize"
arg3 : The new value. You can use a percentage or a value for a number of kilobytes. Units like '5M' or ".5G" can also be used.
