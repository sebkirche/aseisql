Show indexes on a table.

sp_statistics table_name [, table_owner] [, table_qualifier] [, index_name] [, is_unique]

Parameter information :
table_name : The name of a table.
table_owner : The name of the owner of the table. When null the name of the current user is used to look for the table and the dbo.
table_qualifier : The name of the database where the table is located. Can only be null or the name of the current database.
index_name : The name of an index on the table.
is_unique : Show all or only unique indexes. Possible values: N : Show all indexes., Y : Show unique indexes.
