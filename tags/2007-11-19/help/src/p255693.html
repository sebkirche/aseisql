Adds or overwrites a user defined message.

sp_addmessage message_num, message_text [, language [, with_log [,replace]]]

Parameter information :
message_num : Number of the message ( >= 20000).
message_text : Text of the message.
language : Language of the message, the language must exist on the server.
with_log : Outputs the message to the errorlog. Possible values: FALSE, TRUE
replace : Overwrites an existing message/language combination.
