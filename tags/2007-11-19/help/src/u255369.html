Export or import a PKCS #12 file into a certificates file and a private key.

certpk12 {-O Pkcs12_file | -I Pkcs12_file} [-C Cert_file] [-K Key_file] [-P key_password] [-E Pkcs12_password] [-v]

Parameter information :
-O : specifies the name of a PKCS #12 file to be exported.
-I : Specifies the name of a PKCS #12 file to be imported.
-C : Specifies the name of certificate file to be exported to a PKCS #12 file if -O is on; or the name of certificate file to be imported from a PKCS #12 file if -I is on.
-K : Specifies the name of private key file to be exported to a PKCS #12 file if -O is on; or the name of private key file to be imported from a PKCS #12 file if -I is on.
-P : Specifies the password which is used to protect the private key specified by -K.
-E : Specifies the password used to protect the PKCS #12 file.
-v : Print the version number
