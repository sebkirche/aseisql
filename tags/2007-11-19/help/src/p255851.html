Changes a named time range.

sp_modify_time_range name, startday, endday, starttime, endtime

Parameter information :
name : Name of the time range.
startday : Full name of the day of the week to start the range or null to leave the value unchanged.
endday : Full name of the day of the week to end the range or null to leave the value unchanged.
starttime : Time when the range starts in format "hh:mm" or null to leave the value unchanged.
endtime : Full name of the day of the week to end the range or null to leave the value unchanged.
