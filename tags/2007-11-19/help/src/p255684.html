Add a user defined text into the audit trail.

sp_addauditrecord text [,db_name [, obj_name [,owner_name [, dbid [,objid]]]]]

Parameter information :
text : The text to be entered into the audit trail.
db_name : The name of the database.
obj_name : The name of an object.
owner_name : The name of the owner of the object.
dbid : The id-number of the database (db_name).
objid : The object id of the object (obj_name)
