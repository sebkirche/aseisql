Set the cipher suite preferences

sp_ssladmin setciphers, {"FIPS" | "Strong" | "Weak" | "All" | quoted_list}

Parameter information :
FIPS : Fips compliant algoritms (currently ASE, 3DES, DES and SHA1)
Strong : Encription algoritms using keys longer than 64 bits.
Weak : Supported encription algorithms not included in the Strong set.
All : The default set
quoted_list : A comma separated list of cipher suites. See sybase docs for the current list.
