Makes a page in database accessable again after is has been marked suspect by recovery.

sp_forceonline_page dbname, pgid, option [, no_print]

Parameter information :
dbname : Name of the database.
pgid : Number of the page to brought online.
option : Provides access to the page in the database for particular users. Possible values: all_users : Provides access to all users., sa_off : Revokes access from users previously given with sa_on., sa_on : Allows only access for users who have sa_role.
no_print : Do not print remaining suspect pages.
