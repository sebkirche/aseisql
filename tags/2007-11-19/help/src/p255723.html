Show permission information for columns in a table or view.

sp_column_privileges table_name [, table_owner [, table_qualifier [, column_name]]]

Parameter information :
table_name : Name of a table or view.
table_owner : Name of the owner of the table or null to use the current user name and dbo to look for the table.
table_qualifier : The name of the database (use current database name or null).
column_name : Name of a column, wildcards are supported.
