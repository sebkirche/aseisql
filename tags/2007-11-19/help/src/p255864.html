Add a definition for a primary key for a table or view.

sp_primarykey tabname, col1 [, col2 [, col3 [, col4 [, col5 [, col6 [, col7 [, col8]]]]]]]

Parameter information :
tabname : Name for the table or view.
col1 : First column of the key.
col2 : Second column of the key.
col3 : Third column of the key.
col4 : Fourth column of the key.
col5 : Fifth column of the key.
col6 : Sixth column of the key.
col7 : Seventh column of the key.
col8 : Eigth column of the key.
