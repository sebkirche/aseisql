Converts a server certificate request to a CA- (certificate authority) signed certificate.

certauth [-r] [-C caCert_file] [-Q request_filename] [-K caKey_filename] [-O SignedCert_filename] [-P caPassword] [-T valid_time] [-v]

Parameter information :
-r : When specified, creates a self-signed root certificate for the test environment.
-C : Specifies the name of the CA's certificate request file when -r is specified, or specifies the name of the CA's root certificate.
-Q : Specifies the name of certificate request file.
-K : Specifies the name of the CA's private key.
-O : Specifies the name to use for the output when creating a signed certificate file.
-P : Specifies the CA's password that is used to decrypt its private key.
-T : Specifies the valid time range for a signed certificate. The valid time range is in units of days.
-v : Print the version number
