Creates a report with the faults find in a database.

sp_dbcc_faultreport [report_type [, dbname [, objectname [, date [, hard_only [, exclusion_mode [, exclusion_faults [, exclusion_tables [, exclusion_combo [, display_recommendations]]]]]]]]]]

Parameter information :
report_type : The type of report to generate. Possible values: long, short
dbname : The name of a database. When not specified all databases for which faults where found are used.
objectname : The name of table or index to restrict to report to.
date : The date for which a result is generated. When not specified the most recent date is used.
hard_only : Show only hard faults or not Possible values: 0 : hard faults only (default), 1 : Soft and hard faults
exclusion_mode : Indicate how persistent errors are handled. Possible values: extend : Applies the temporary exclusion list as well as the persistent exclusion list (exclusion faults), ignore : Ignores the persistent exclusion list and uses the temporary exclusion list (exclusion faults)
exclusion_faults : A comma separated list of faults to be excluded from reporting.
exclusion_tables : A comma separated list of tables to be excluded from reporting.
exclusion_combo : A comma separated list of faults and tables to be excluded from reporting.
display_recommendations : Show recommendations for fixes. Possible values: 0 : Dot not show (default), 1 : Show recommendations.
