Interactive SQL parser to ASE.

isql [-b] [-e] [-F] [-p] [-n] [-v] [-X] [-Y] [-Q] [-a display_charset] [-A packet_size] [-c cmdend] [-D database] [-E editor] [-h headers] [-H hostname] [-i inputfile] [-I interfaces_file] [-J client_charset] [-K keytab_file] [-l login_timeout] [-m errorlevel] [-o outputfile] [-P password] [-R remote_server_principal] [-s colseparator] [-S server_name] [-t timeout] -U username [-V [security_options]] [-w columnwidth] [-z locale_name] [-Z security_mechanism]

Parameter information :
-b : disables the display of the table headers output.
-e : echoes input.
-F : enables the FIPS flagger. When you specify the -F parameter, the server returns a message when it encounters a non-standard SQL command.
-p : prints performance statistics.
-n : removes numbering and the prompt symbol (>) from the echoed input lines in the output file when used in conjunction with -e.
-v : prints the version number
-X : initiates the login connection to the server with client-side password encryption.
-Y : tells the ASE to use chained transactions.
-Q : provides clients with failover property.
-a : runs isql from a terminal whose character set differs from that of the machine on which isql is running.
-A : specifies the network packet size to use for this isql session.
-c : changes the command terminator (default = go)
-D : selects the database in which the isql session begins.
-E : specifies an editor other than the default editor vi.
-h : specifies the number of rows to print between column headings.
-H : sets the client host name.
-i : specifies the name of the operating system file to use for input to isql.
-I : specifies the name and location of the interfaces file to search when connecting to ASE.
-J : specifies the character set to use on the client.
-K : specifies the path to the keytab file used for authentication in DCE.
-l : specifies the maximum timeout value allowed when connecting to ASE.
-m : customizes the error message display. For errors of the severity level specified or higher, only the message number, state, and error level are displayed; no error text appears.
-o : specifies the name of an operating system file to store the output from isql.
-P : specifies your ASE password.
-R : specifies the principal name for the server as defined to the security mechanism.
-s : resets the column separator character, which is blank by default.
-S : specifies the name of the ASE to which to connect.
-t : specifies the number of seconds before a SQL command times out.
-U : specifies a login name.
-V : specifies network-based user authentication.
security_options : enables additional security services. Possible values: c : Enable data confidentiality service, i : Enable data integrity service, m : Enable mutual authentication for connection establishment, o : Enable data origin stamping service, q : Enable out-of-sequence detection, r : Enable data replay detection
-w : sets the screen width for output. The default is 80 characters.
-z : is the official name of an alternate language to display isql prompts and messages.
-Z : specifies the name of a security mechanism to use on the connection.
