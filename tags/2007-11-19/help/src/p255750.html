Store or delete a configuration setting for a certain database or server-wide.

sp_dbcc_updateconfig dbname, type, str1 [, str2]

Parameter information :
dbname : The name of a database. When null is used here, the configuration value is used for all databases.
type : The name of a configuration parameter. Possible values: dbcc named cache : The name of a cache to use (str1) and the size in pages (str2)., enable automatic workspace expansion : When set to "1" (str1) the workspace is expanded when needed., IO error abort : The error count (str1)., linkage error abort : The linkage error count (srt1)., max worker processes : The maximum number of worker processes to use (str1), OAM count threshold : The OAM count (str1), scan workspace : The name of a scan workspace (srt1)., text workspace : The name of a text workspace (str1).
str1 : The first value for the parameter. You can also specify "delete" to remove a configuration value.
str2 : The second value for the parameter (where applicable).
