Show tables, views or system tables.

sp_tables [table_name] [, table_owner] [, table_qualifier] [, table_type]

Parameter information :
table_name : The name of a table. Wildcards can be used.
table_owner : Name of the owner of the table. Wildcards can be used.
table_qualifier : The name of the database where the table should be located. Must be null or the name of the current database.
table_type : A list of values enclosed with double quotes, separated by commas. Possible values: 'SYSTEM TABLE', 'TABLE', 'VIEW'
