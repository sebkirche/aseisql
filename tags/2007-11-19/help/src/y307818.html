Configure a primary companion

sp_companion remoteserver, configure [, option] [, srvlogin] [, srvpwd] [, cluslogin] [, cluspwd]

Parameter information :
remoteserver : Name of the server to be configured as a primary companion.
option : Option to configure. Possible values: null : Do not create proxy databases at the companion server., with_proxydb : Creates proxy databases and proxy tables for all non-system databases
srvlogin : User name to login on the companion server.
srvpwd : Password to login.
cluslogin : Login name to logon to the cluster.
cluspwd : Password to logon to the cluster.
