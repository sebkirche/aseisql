To stop sampling

sp_sysmon end_sample [ ,section] [ ,applmon] [ ,filter] [, dumpcounters]

Parameter information :
section : Reporting section or null for all sections. Possible values: appmgmt : Application management, cache wizard, dcache : Data cache management, diskio : Disk I/O management, esp : Extended stored procedure management., housekeeper, indexmgmt : Index management, kernel : Kernel utilisation, locks : Lock management, mdcache : Metadata cache management., memory : Memory management, monaccess : Monitor access, netio : Network I/O, pache : Procedure cache management, parallel : Parallel query management, recovery : Recovery management, repagent, taskmgmt : Task management, wpm : Worker process management, xactmgmt : Transaction management, xactsum : Transaction profile
applmon : Details per application or login or not detailed to that level. Possible values: appl_and_login : CPU, I/O, priority changes, resource limits per application name and login anme, appl_only : CPU, I/O, priority changes, resource limits per application name, no_appl : No detail per application / login.
filter : The name of a cache (can only be used when section = 'cache wizard')
dumpcounters : Dump the raw monitoring values. Possible values: n : (default), y
