Binds a database or an object to a cache.

sp_bindcache cachename, dbname [, tablename [, indexname]]

Parameter information :
cachename : Name of an existing cache.
dbname : Name of the database to be bound to cache, or where the object to bound is located.
tablename : Name of the table to be bound (user name can be included).
indexname : Name of the index to be bound. Possible values: "text only" : Binds text or image objects to a cache.
