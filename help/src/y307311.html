Convert an expression to another datatype

convert(datatype [(length) | (precision [, scale])] [null | not null], expression)

Parameter information :
datatype : Any system defined datatype or a Java class to convert the expression into. User defined datatypes are not allowed here.
length : Length for the datatype, when appropriate. 30 is a default for char and binary types.
precision : Precision (for float, numeric, decimal datatype)
scale : Scale, for numeric and decimal datatype.
expression : The expression to convert.
