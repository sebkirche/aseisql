Creates, using resource files, a new Adaptive Server, Backup Server, Monitor Server, or XP Server with default or user-specified values for key configuration attributes.

srvbuildres [-s sybase_dir] [-I interfaces_file] [-r resource_file] [-v]

Parameter information :
-s : specifies the value to use for the SYBASE environment variable.
-I : specifies the name and location of the interfaces file to search when connecting to ASE.
-r : executes the specified resource file.
-v : Print the version number.
