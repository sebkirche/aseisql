Manage the last change threshold

lct_admin({"lastchange" | "logfull" | "reserved_for_rollbacks"}, database_id)

Parameter information :
lastchange : Create a last change threshold for the specified database.
logfull : Returns 1 if the last change threshold has been crossed in the specified database and 0 if it has not.
reserved_for_rollbacks : Determines the number of pages in a database currently reserved for rollbacks.
database_id : The db-id of the database.
