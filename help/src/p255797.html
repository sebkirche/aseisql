Show foreign key constraints.

sp_fkeys pktable_name [, pktable_owner] [, pktable_qualifier] [, fktable_name] [, fktable_owner] [, fktable_qualifier]

Parameter information :
pktable_name : Name of the primary key table. Specify pktable_name or fktable_name or both.
pktable_owner : Owner of the primary table. When not specified the current user name and dbo are used to look for the table.
pktable_qualifier : Name of the database for the primary table. Must be null or the name of the current database.
fktable_name : Name of the foreign key table. Specify pktable_name or fktable_name or both.
fktable_owner : Owner of the foreign key table. When not specified the current user name and dbo are used to look for the table.
fktable_qualifier : Name of the database for the foreign key table. Must be null or the name of the current database.
