Drops a resource limit.

sp_drop_resource_limit name, appname [, rangename, limittype, enforced, action, scope]

Parameter information :
name : The name of the login to which the limit applies or null for all users.
appname : The name of the application to which the limit applies (or null for all applications).
rangename : The timerange for the limit or null for all limits.
limittype : Type of the limit. Possible values: elapsed_time, io_cost, null : For all limits regardsless of their type., row_count, tempdb_space
enforced : Type of enforcement for the limit. Possible values: 1 : Prior (only for io cost)., 2 : During execution., 3 : Prior and during execution., null : Regardless of the type of enforcement.
action : The type of action to be taken when the limit has been reached. Possible values: 1 : Warning, 2 : Abort the batch, 3 : Abort transaction., 4 : Kill the session, null : Regardless of the action.
scope : Type of scope. Possible values: 1 : Queries., 2 : Query batches., 4 : Transactions, 6 : Query batches and transactions., null : Regardless of the scope.
