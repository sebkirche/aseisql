Executed when the last change threshold is crossed and this procedure is created. Can also be given another name.

sp_thresholdaction dbname, segment_name, space_left, status

Parameter information :
dbname : Name of the database where the segment resides.
segment_name : Name of the segment.
space_left : The size of the threshold in pages.
status : Type of threshold. Possible values: -1 : Last change threshold, 0 : Normal threshold
