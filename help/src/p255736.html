Create a workspace for the dbccdb database.

sp_dbcc_createws dbname, segname [, wsname] , wstype, "wssize"

Parameter information :
dbname : Specifiy the name of a dbcc database.
segname : The name of a segment for the workspace.
wsname : The name for a workspace. When no value is given (null) a name is generated.
wstype : Type of workspace. Possible values: scan, text
wssize : The size of workspace. Specifiy a number to indicate pages, or followed by "K" (kiloBytes) or "M" (megaBytes)
