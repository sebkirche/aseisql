Loads the character sets and sort order files in ASE.

charset [-P password] [-S server] [-I interface] sort_order [ characterset ]

Parameter information :
-P : Specifies your password.
-S : Specifies the name of the server on which to change the character set and sort order.
-I : Specifies the network interface used by the server.
sort_order : Specifies the name of the sort order file Adaptive Server will use.
characterset : Specifies the character set Adaptive Server will use.
