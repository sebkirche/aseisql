Send an instruction to BackupServer during a dump or load.

sp_volchanged session_id, devname, action [, fname [, vname]]

Parameter information :
session_id : Name of the session within Backup Server.
devname : When changing a volume the name of the device where the new volume is located.
action : Action for the Backup Server. Possible values: abort, proceed, retry
fname : Name of the file on the tape.
vname : Name of the tape volume.
