Create or modify a user defined execution class.

sp_addexeclass classname, priority, timeslice, engine_group

Parameter information :
classname : Name of the execution class.
priority : The priority for the execution class. Possible values: HIGH : High priority, LOW : Low priority, MEDIUM : Medium priority
timeslice : This parameter is ignored
engine_group : The name of an existing engine group. Possible values: ANYENGINE : The execution class can run on any engine.
