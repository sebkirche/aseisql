Create a pool or change its size

sp_poolconfig cachename, parm1, parm2, parm3

Parameter information :
cachename : Name of an existing cache.
parm1 : The size of the pool in multiples of the following: Possible values: G : GBytes, K : KBytes (default), M : MBytes, P : Pages
parm2 : IO size for the pool. Specify a value of units of the logical page size (maximum 4 times), followed by the letter K. Possible values: 16K, 2K, 4K, 8K
parm3 : The pool where the space should be taken from / given back to. All the space in a cache is allocated to certain pool so a new pool or change in the size of the pool affects always another pool.
