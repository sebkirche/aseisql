Manage the persistent exclusion list, used by dbcc checkverify and sp_dbcc_faultreport.

sp_dbcc_exclusions dbname, op, type, exclusion_list

Parameter information :
dbname : The name of a database or null for all databases.
op : Action to perform. Possible values: add : Create a new exclusion, drop : Remove an exclusion, listall : List the exclusions.
type : The type of object specified in the exclusion list. Possible values: combo : The exclusion list contains a table name combined with a fault number., faults : The exclusion_list must contain fault numbers., tables : The exclusion list must contain table names.
exclusion_list : A comma separated list of faults, tables or a combination to set a fault for a particular table. Use format table_name:fault_number.
