Send a message. Windows NT only.

xp_sendmail recipient [, subject] [, cc_recipient] [, bcc_recipient] [, query] [, message] [, attachname] [, attach_result] [, echo_error] [, include_file] [, no_column_header] [, width] [, separator] [, dbuser] [, dbname] [, type] [, include_query]

Parameter information :
recipient : A semicolon separated list of the recipients of the message.
subject : Subject of the message.
cc_recipient : A semicolon separated list of the cc's.
bcc_recipient : A semicolon separated list of the bcc's.
query : A sql query to execute and the results are then the attachment or the message body of the message.
message : The text of the message. Use query or message, not both.
attachname : File name of the attachment where the results of the query are put into.
attach_result : How the results are combined with the message. Possible values: false : Within the message body., true : As attachment.
echo_error : Include SQL error messagesand rowcount. Possible values: false, true
include_file : A semicolon separated list with filename to be used as attachments.
no_column_header : Sent column headings with queries. Possible values: false, true
width : Width of the result set.
separator : Column separator for queries.
dbuser : Database user to be used when executing the query. guest is default.
dbname : Name of the database where the query should be executed.
type : Type of message. Possible values: CMC:IPM, null : Defaults to CMC:IPM
include_query : Should the query be appended to the result set. Possible values: false, true
