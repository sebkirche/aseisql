Returns the name for a column of an index on the specified table or view.

index_col (object_name, index_id, key_# [, user_id])

Parameter information :
object_name : The name of a table or view.
index_id : The id of the index. Use the same value as sysindex.indid for the specified table.
key_# : The number of a key (column) within the index. Range is between 1 and sysindexes.keycnt (clustered index) or sysindexes.keycnt + 1 (nonclustered index).
user_id : The id of the owner of the table. Defaults to your own user_id.
