Manages remote login options.

sp_remoteoption [remoteserver [, loginame [, remotename [, trusted [, optvalue]]]]]

Parameter information :
remoteserver : Name of the remote server
loginame : Name of the local login to where the remote logins are mapped to.
remotename : Name of the user on the remote server.
optvalue : Toggles trusted mode. Possible values: false : Passwords are verified., true : Passwords are not checked or needed when logging on.
