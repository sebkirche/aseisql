Creates a time range.

sp_add_time_range name, startday, endday, starttime, endtime

Parameter information :
name : Name of the time range.
startday : Full name of the day of the week to start the range.
endday : Full name of the day of the week to end the range.
starttime : Time when the range starts in format "hh:mm".
endtime : Time when the range ends in format "hh:mm".
