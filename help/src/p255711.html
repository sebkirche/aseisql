Configures the cache strategy for individual objects.

sp_cachestrategy dbname, tablename, indexname, strategy, onoff

Parameter information :
dbname : Name of the database where the object is located.
tablename : Name of the table.
indexname : Name of the index or one of the following values: Possible values: "table only" : For a table., "text only" : For text / image objects.
strategy : Strategy for the object. Possible values: mru, prefetch
onoff : Value for the strategy. Possible values: "off", "on"
