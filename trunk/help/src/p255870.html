Executes a query on a remote server.

sp_remotesql server, query [, query2, [ query-n] ...]]

Parameter information :
server : Name of the server to execute the query on.
query : A sql statement. The statement is passed to a varchar(255) so this limits the size of the query, but you can specify up to 254 of those varchars.
query2 : Second line of the query.
query-n : N-th line of the query.
