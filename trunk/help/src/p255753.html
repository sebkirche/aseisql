Sets the sequence in which user database are recovered.

sp_dbrecovery_order [dbname [, rec_order [, force [, mode]]]]

Parameter information :
dbname : Name of a user database
rec_order : Sequence within the recovery process. Specify a -1 to delete the database from the user defined recovery sequence.
force : When specified, inserts the database at the specified place in stead of at the end.
mode : Set a strict or relaxed mode. Possible values: "relax" : (default setting), "strict"
