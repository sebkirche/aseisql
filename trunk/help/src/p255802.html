Defines a foreign key on a table or view.

sp_foreignkey tabname, pktabname, col1 [, col2 [, col3 [, col4 [, col5 [, col6 [, col7 [, col8]]]]]]]

Parameter information :
tabname : Name of the table or view where the foreign key is located.
pktabname : Name of the table or view where the primary key is located.
col1 : Name of the first column of the foreign key.
col2 : Name of the second column of the foreign key.
col3 : Name of the third column of the foreign key.
col4 : Name of the fourth column of the foreign key.
col5 : Name of the fifth column of the foreign key.
col6 : Name of the sixth column of the foreign key.
col7 : Name of the seventh column of the foreign key.
col8 : Name of the eigth column of the foreign key.
