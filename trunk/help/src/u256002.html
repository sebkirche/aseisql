Historical Server

histserver -D homeDir -d delim [-I interfacesFile] [-l logFile] [-U userName] [-P password] [-n maxConnections] [-S hsName] [-v] [-O ASEname] [-o database] [-u ASEuser] [-p ASEpassword]

Parameter information :
homeDir : The home directory for the server.
delim : A file delimiter for the files generated
interfacesFile : Path name of the interfaces file
logFile : Log file for the server (defaults to hs.log)
userName : Name of the superuser for the historical server.
password : Password for the superuser
maxConnections : Maximum number of client connections to the historical server.
hsName : Name of the historical server. Must be present in the interfaces file.
-v : Show version information
ASEname : Name of an ASE where the output should be written to.
database : Name of the database where the output should be written to, defaults to hs_monitoring.
ASEuser : Name of a login for the output ASE server.
ASEpassword : Password for the login for the output ASE server.
