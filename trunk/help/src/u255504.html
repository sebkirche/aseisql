Copies a retained JAR and the classes it contains from an ASE into a client file.

extractjava -j jar_name -f file_name [-S server_name] [-U user_name] [-P password] [-D database_name] [-I interfaces_file] [-a display_charset] [-J client_charset] [-z language] [-t timeout] [-v]

Parameter information :
-j : the name assigned to the retained JAR in the database that is the source of the transfer.
-f : the name of the client file that is the target of the transfer.
-S : the name of the server.
-U : an ASE login name.
-P : an ASE password.
-D : the name of the database in which to install the JAR.
-I : the name and location of the interfaces file to search when connecting to ASE.
-a : allows you to use extractjava from a machine where the character set differs that of the server.
-J : specifies the character set to use on the client.
-z : the name of an alternate language for displaying extractjava prompts and messages.
-t : specifies the number of seconds before a SQL command times out.
-v : prints the version number
