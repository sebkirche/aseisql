Creates a summary report for a checkstorage operation.

sp_dbcc_summaryreport [dbname [, date [, op_name [, display_recommendations]]]]

Parameter information :
dbname : The name of a database.
date : The date to use for which results are available. When not given the most recent data is used.
op_name : The kind of operation. Possible values: checkstorage, checkverify
display_recommendations : Show recommendations for fixes or not. Possible values: 0 : Do not show (default), 1 : Show
