Read a message from the mailbox. Windows NT only.

xp_readmail [msg_id] [, recipients output] [, sender output] [, date_received output] [, subject output] [, cc output] [, message output] [, attachments output] [, suppress_attach] [,peek] [, unread] [, msg_length output] [, bytes_to_skip [output]] [, type [output]]

Parameter information :
msg_id : Identifier of the message.
recipients : A semicolon separated list of the recipients of the message.
sender : Sender of the message.
date_received : Date of receiving the message.
subject : Subject of the message.
cc : A semicolon separated list of the cc's.
message : The message text.
attachments : A semicolon separated list with the locations of the attachment(s).
suppress_attach : Do not create temporary files for attachments. Possible values: false, true : (Default)
peek : Set the message as read. Possible values: false : Set to read., true : Set to unread.
unread : Consider only unread messages. Possible values: false, true
msg_length : The total length of the message.
bytes_to_skip : Use this to read messages longer than 255 bytes to read messages with several calls of xp_readmail.
type : Type of the message. Possible values: CMC:IPM, null : Defaults to CMC:IPM
