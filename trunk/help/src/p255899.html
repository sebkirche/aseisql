Show column information for the optimum unique index or a timestamp column.

sp_special_columns table_name [, table_owner] [, table_qualifier] [, col_type]

Parameter information :
table_name : Name of a table or view.
table_owner : The owner of the table. When not specified the name of the current user is used and then dbo to look for the table.
table_qualifier : The name of the database for the table. Use only null or the name of the current database.
col_type : Used to specify which information to return. Possible values: null : Show everything., R : Show uniqueness information., V : Show only timestamp columns
