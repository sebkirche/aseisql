Binds an execution class to an application, login or stored procedure.

sp_bindexeclass object_name, object_type, scope, classname

Parameter information :
object_name : Name of the application, login or stored procedure.
object_type : Identifies object name. Possible values: ap : Application, lg : Login, pr : Stored procedure
scope : Sets the scope when object_type is a stored procedure. Enter the name of an application or login.
classname : The class to bind the object to. Can be a user defined class, or: Possible values: ANYENGINE, EC1, EC2, EC3
