Makes an index accessable again after is has been marked suspect by recovery.

sp_forceonline_object dbname, objname, indid, option [, no_print]

Parameter information :
dbname : The name of a database.
objname : The name of a table.
indid : The number of an index of the table
option : Provides access to the index on the table for particular users. Possible values: all_users : Provides access to all users., sa_off : Revokes access from users previously given with sa_on., sa_on : Allows only access for users who have sa_role.
no_print : Do not print a list of other objects that are still suspects.
