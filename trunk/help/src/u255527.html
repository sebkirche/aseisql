Installs a JAR from a client file into an ASE.

installjava -f file_name [ -new | -update ] [ -j jar_name ] [ -S server_name ] [ -U user_name ] [ -P password ] [ -D database_name ] [ -I interfaces_file ] [ -a display_charset ] [ -J client_charset ] [ -z language ] [ -t timeout ] [-v]

Parameter information :
-f : the name of the source file containing the classes to be installed in the database.
-new : specifies whether the classes in the file already exist in the database. If you specify the new parameter, you cannot install a class with the same name as an existing class.
-update : specifies whether the classes in the file already exist in the database. If you specify the update parameter, you can install a class with the same name as an existing class, and the newly installed class replaces the existing class.
-j : the name of the JAR containing the classes to be installed in the database.
-S : the name of the server.
-U : an ASE login name.
-P : an ASE password.
-D : the name of the database in which to install the JAR.
-I : the name and location of the interfaces file to search when connecting to ASE.
-a : allows you to use installjava from a machine where the character set differs that of the server.
-J : specifies the character set to use on the client.
-z : the name of an alternate language for displaying installjava prompts and messages.
-t : specifies the number of seconds before a SQL command times out.
-v : Print the version number.
