Change a specific option for a database.

sp_dboption [dbname, optname, optvalue [, dockpt]]

Parameter information :
dbname : Name of the database
optname : Name of the option. You can specify starting characters to uniquely identify any option. Possible values: abort tran on log full, allow nulls by default, async log service : Use the asynchronous log service only for 4 or more engines., auto identity, dbo use only, ddl in tran, disable alias access, identity in nonunique index, no chkpt on recovery, no free space acctg, read only, select into/bulkcopy/pllsort, single user, trunc log on chkpt, trunc. log on chkpt., unique auto_identity index
optvalue : Value of the setting. Possible values: false : Off., true : On.
dockpt : Control whether an automatic checkpoint will be given Possible values: 0 : Don't do a checkpoint, 1 : Checkpoint automatically (default)
