Set a rule for database expansion

sp_dbextend "set","database", db_devname, arg1 {[, arg2] [, arg3]}

Parameter information :
db_devname : The name of a database or 'default' for a server wide setting.
arg1 : The name of a segment.
arg2 : The amount of space the database can grow by. You can use a percentage or a unit specifier (like "100M").
arg3 : The maximum size of the segment. No further expansion will occur when this size has been reached.
