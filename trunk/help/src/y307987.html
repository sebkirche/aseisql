Change the range or total density for a column(s)

sp_modifystats objname, column_group, MODIFY_DENSITY, opt_param1, opt_param2, opt_param3

Parameter information :
objname : The name of the table. Can include database and/or owner.
column_group : The name(s) of the columns separated with a comma. A trailing wildcard % can be used. Possible values: % : Use all column names., all : Modifies all column group for this table. Because 'all' is a keyword, it requires quotes.
opt_param1 : Specify which density should be changed. Possible values: range : Modifies the range cell density, total : Modifies the total cell density
opt_param2 : Specify what should by done with the current statistical value. Possible values: absolute : Ignore the current value and use the number specified by the opt_param3 parameter, factor : Multiply the current statistical value by the opt_param3 parameter
opt_param3 : Is either the specified density value or a multiple for the current density. Must be between zero and one, inclusive, if absolute is specified.
