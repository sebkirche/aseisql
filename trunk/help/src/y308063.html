Bind a login or application to a temporary database or the default group

sp_tempdb "bind", arg1, arg2, arg3, arg4, null, arg6

Parameter information :
arg1 : The type of object to to bind. Possible values: AP : Short for application_name., application_name, LG : Short for login_name, login_name
arg2 : The name of the application or login.
arg3 : Bind the login/application (arg2) to either a group or a database Possible values: database, DB : Short for database, GR : Short for group, group
arg4 : The name of database (arg3 must be DB or database) or the name of a group (must always be "default").
arg6 : Specify the hardness of the binding. Possible values: hard : When the assignment of the login/application to the temporary database/group fails, the login process will not proceed., null : Defaults to 'soft' binding., soft : When the assignment of the login/application to the temporary database/group fails, the system temporary database will be used.
