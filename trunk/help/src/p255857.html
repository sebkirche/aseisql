Shows information about locks being held during a certain interval.

sp_object_stats interval [, top_n [, dbname, objname [, rpt_option]]]

Parameter information :
interval : The time thats locks should be monitored (format "hh:mm:ss")
top_n : Number of objects to be shown on the report.
dbname : The name of specific database to monitored.
objname : The name of a table in the database to be monitored.
rpt_option : Type of report. Possible values: rpt_locks : Shows tables with the highest contention., rpt_objlist : Shows tables with the highest lock activity.
