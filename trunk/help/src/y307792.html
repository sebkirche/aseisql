Configure auditing

sp_audit option, login_name, object_name [,setting]

Parameter information :
option : What should be audited. Possible values: adhoc, all, alter, bcp, bind, cmdtext, create, dbaccess, dbcc, delete, disk, drop, dump, errors, exec_procedure, exec_trigger, func_dbaccess, func_obj_access, grant, insert, install, load, login, logout, mount, quiesce, reference, remove, restart, revoke, rpc, security, select, setuser, table_access, truncate, unbind, unmount, update, view_access
login_name : Name of a login or optionally when option = all the name of a system defined role.
object_name : The name of a particular object or : Possible values: all, default procedure, default table, default trigger, default view
setting : The level of auditing. Possible values: fail, off, on, pass
