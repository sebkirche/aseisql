Creates a new language and sets the date format, months and days.

sp_addlanguage language, alias, months, shortmons, days, datefmt, datefirst

Parameter information :
language : Name of the language.
alias : An alias name or null.
months : A comma separated list with the full names of the months.
shortmons : A comma separated list with the abbreviated names of the months.
days : A comma separated list with the names of the days.
datefmt : A format definition for entering dates. Possible values: dmy, dym, mdy, myd, ydm, ymd
datefirst : The number of the first day of week. Possible values: 1 : Monday, 2 : Tuesday, 3 : Wednesday, 4 : thursday, 5 : Friday, 6 : Saturday, 7 : Sunday
