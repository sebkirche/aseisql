Creates a dump device.

sp_addumpdevice devtype, logicalname, physicalname [, tapesize]

Parameter information :
devtype : Type of the device. Possible values: "disk" : For disks or files., "tape" : For tape drives.
logicalname : Name of the dump device.
physicalname : The physical name (location) of the actual device.
tapesize : Size of the tape in Mb.
