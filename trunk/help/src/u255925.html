Installs and modifies languages, character sets, and sort order defaults for ASE, using a resource file.

sqllocres [-S Server] [-U User] [-P Password] [-s SybaseDir] [-I Interfacesfile] [-r Resourcefile] [-v]

Parameter information :
-S : specifies the name of the ASE to which to connect.
-U : specifies a login name.
-P : specifies the password.
-s : specifies the value to use for the SYBASE environment variable.
-I : specifies the name and location of the interfaces file to search when connecting to ASE.
-r : executes the specified resource file.
-v : Print the version number.
