Show permissions for database objects, users, groups or roles.

sp_helprotect [name [, username [, option [, rolename]]]]

Parameter information :
name : The name of a table, view, stored procedure, user, group or user defined role.
username : The name of a user.
option : The level of detail for the information. Possible values: enabled : Includes information on all roles activated by the user when determining permissions granted., grant : Show the privileges grant to the specified user (parameter username) "with grant option"., granted : Includes information on all roles granted to the user when determining permissions granted., none : Ignore roles granted to the user when determining permissions granted.
rolename : Display permission information for the specified role, regardless of whether the role has been granted to the user.
