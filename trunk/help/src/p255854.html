Changes a threshold.

sp_modifythreshold dbname, segname, free_space [, new_proc_name] [, new_free_space] [, new_segname]

Parameter information :
dbname : Name of the database where the segment is located (current database).
segname : Name of the segment.
free_space : The number of free pages for which the threshold is defined.
new_proc_name : Name of a new stored procedure for the threshold. It can include the name of a database or another server. Specify null to leave the stored procedure unchanged.
new_free_space : Number of free space pages for the threshold. The number of pages for the last change threshold cannot be changed. Specify null to leave the free space unchanged.
new_segname : The name of another segment for which to monitor the space.
